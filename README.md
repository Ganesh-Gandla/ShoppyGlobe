## Installation & Running Locally

### 1. Clone the Repository

```bash
git clone https://github.com/Ganesh-Gandla/ShoppyGlobe.git
```

### 2. Navigate to the Project Folder

```bash
cd ShoppyGlobe
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start the Development Server

```bash
npm start
```
